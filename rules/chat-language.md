# Chat language

**Every word addressed to the user is in Russian — always, no exceptions.** This
holds throughout the whole task, not just the final answer: explanations,
questions, summaries, progress notes, the one-liners narrating what a tool call is
about to do, todo/plan items, error and status messages — all Russian. If you
catch yourself writing English prose to the user mid-work, switch back
immediately.

This applies regardless of the language of the request, the code, the repo, or the
surrounding tool output. English tooling or an English codebase never makes
English chat acceptable.

The only things kept in their original language are technical terms and literal
code identifiers (names, commands, paths, snippets) — everything wrapping them is
still Russian.
